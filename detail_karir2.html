Deskripsi Pekerjaan

Kami mencari Back End Developer berpengalaman untuk bergabung dengan tim teknologi PT. Tulungagung Jaya. Anda akan bertanggung jawab untuk mengembangkan, memelihara, dan mengoptimalkan sistem back end untuk aplikasi bisnis utama perusahaan.

Tanggung Jawab Utama

  • Mengembangkan dan memelihara RESTful API menggunakan Node.js
  • Mendesain dan mengoptimalkan database PostgreSQL
  • Menerapkan sistem autentikasi dan otorisasi yang aman
  • Mengintegrasikan dengan layanan third-party dan microservices
  • Menulis kode yang bersih, efisien, dan terdokumentasi
  • Melakukan optimasi performa dan scaling aplikasi
  • Bekerja sama dengan tim front end untuk integrasi API
  • Menerapkan praktik keamanan data dan privasi

Kualifikasi Teknis

Wajib:
  • Pengalaman 2+ tahun dengan Node.js
  • Mahir dalam JavaScript/TypeScript
  • Pengalaman dengan PostgreSQL/MongoDB
  • Memahami konsep RESTful API dan GraphQL
Dianjurkan:
  • Pengalaman dengan Docker dan Kubernetes
  • Pengetahuan tentang sistem terdistribusi
  • Familiar dengan cloud platform (AWS/GCP)
  • Pengalaman dengan sistem message queue (RabbitMQ/Kafka)
