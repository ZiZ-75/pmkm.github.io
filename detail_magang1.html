Magang

Lowongan Magang

Magang Front End Developer

PT. Bhinneka Menteng, Jakarta

Durasi: 3-6 bulan (full-time/hybrid)

Kuota: 5 peserta

Deskripsi Magang

Program magang ini dirancang untuk mahasiswa/peserta yang ingin mengembangkan keterampilan pengembangan web front end dalam lingkungan e-commerce skala besar. Peserta akan bekerja langsung dengan tim developer Bhinneka untuk membangun dan mengoptimalkan antarmuka pengguna (UI) yang responsif, interaktif, dan sesuai kebutuhan bisnis.

Tanggung Jawab

  • Membuat komponen web menggunakan HTML, CSS, JavaScript, dan framework seperti React.js/Vue.js
  • Mengimplementasikan desain dari tim UI/UX ke kode yang efisien
  • Meningkatkan kecepatan loading dan responsivitas website
  • Melakukan testing cross-browser dan cross-device
  • Berkolaborasi dengan back end developer dan desainer
  • Berkontribusi pada pengembangan fitur baru atau perbaikan website Bhinneka

Persyaratan

Teknikal:
  • Pemahaman kuat HTML5, CSS3, JavaScript (ES6+)
  • Pengalaman dengan React.js/Vue.js (nilai plus)
  • Familiar dengan REST API dan version control (Git)
Non-Teknikal:
  • Mahasiswa aktif (D3/S1) jurusan IT atau bidang terkait
  • Kemampuan komunikasi dan kerja tim
  • Kreatif dan detail-oriented

Manfaat Magang

Bimbingan langsung dari senior developer
Sertifikat setelah menyelesaikan magang
Kesempatan terhubung dengan profesional IT
Peluang untuk direkrut sebagai karyawan tetap

Proses Seleksi

1
Pendaftaran

Kirim CV dan portofolio proyek front end (GitHub/tautan website)

2
Tes Teknis

Online coding test (HTML/CSS/JS)

3
Wawancara

Wawancara dengan tim HR dan engineering
