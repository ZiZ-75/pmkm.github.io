Magang

Lowongan Magang

Magang Fullstack Developer

PT. Reog Kendang, Ponorogo

Durasi: 5 bulan (full-time)

Kuota: 4 peserta

Mulai: 1 September 2024

Deskripsi Magang

Program magang intensif untuk mengembangkan keterampilan fullstack development dari dasar hingga mahir. Peserta akan mendapatkan pengalaman langsung mengembangkan aplikasi web lengkap mulai dari frontend hingga backend, dengan bimbingan mentor profesional di industri.

Tech Stack yang Akan Dipelajari:

React.js

Node.js

PostgreSQL

Express.js

Tanggung Jawab

Frontend:
  • Membangun komponen UI dengan React.js
  • Implementasi desain responsif
  • State management dengan Redux
Backend:
  • Membangun RESTful API dengan Node.js
  • Desain dan implementasi database
  • Autentikasi dan otorisasi

Persyaratan

Teknikal:
  • Pemahaman dasar HTML, CSS, dan JavaScript
  • Pengalaman dengan framework JS (nilai plus)
  • Memahami konsep dasar back end
Non-Teknikal:
  • Mahasiswa aktif TI/Sistem Informasi
  • Kemampuan belajar mandiri
  • Komunikasi yang baik

Manfaat Magang

Proyek nyata untuk portofolio
Mentoring 1-on-1 dengan developer senior
Sertifikat + surat rekomendasi
Tunjangan bulanan Rp 1.500.000
Makan siang + snack
Peluang kerja setelah magang

Kurikulum Pembelajaran

Pengenalan ekosistem fullstack, Git workflow, dasar-dasar React dan Node.js
Membangun aplikasi fullstack sederhana, integrasi frontend-backend, dasar database
Pengembangan proyek nyata perusahaan dalam tim, agile methodology, deployment

Proses Seleksi

1
Pendaftaran

Kirim CV, transkrip, dan jawaban technical questionnaire

2
Technical Test

Online coding test (JavaScript dasar)

3
Interview

Technical interview + culture fit
