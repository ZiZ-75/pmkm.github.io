Deskripsi Pekerjaan

Kami mencari Front End Developer berbakat untuk bergabung dengan tim pengembangan produk digital PT. Bhinneka. Anda akan bertanggung jawab untuk membangun antarmuka pengguna yang menarik dan responsif untuk platform e-commerce kami yang melayani jutaan pengguna setiap bulan.

Tanggung Jawab

  • Mengembangkan antarmuka pengguna baru menggunakan React.js
  • Mengoptimalkan komponen untuk performa maksimum di berbagai perangkat dan browser
  • Berkolaborasi dengan tim desain untuk menerapkan mockup menjadi kode yang efisien
  • Menerapkan praktik terbaik pengembangan web modern (responsive design, aksesibilitas)
  • Bekerja sama dengan tim back end untuk integrasi API
  • Menulis kode yang bersih, terdokumentasi, dan dapat dipelihara
  • Berpartisipasi dalam code review dan diskusi teknis

Persyaratan

Teknikal:
  • Pengalaman 1+ tahun sebagai Front End Developer
  • Mahir dalam HTML5, CSS3, dan JavaScript ES6+
  • Pengalaman dengan React.js dan state management (Redux/Context API)
  • Memahami konsep RESTful API dan integrasi dengan front end
  • Pengalaman dengan tools seperti Git, Webpack, Babel
  • Familiar dengan testing framework (Jest, Enzyme, dll)
Non-Teknikal:
  • Pendidikan minimal D3/S1 bidang Ilmu Komputer atau terkait
  • Kemampuan komunikasi dan kerja tim yang baik
  • Orientasi pada detail dan passion untuk UX yang baik
  • Kemampuan memecahkan masalah secara mandiri

Teknologi yang Digunakan

React.js JavaScript ES6+ HTML5 CSS3/SASS Git Redux Responsive Design

Benefit

  • Gaji kompetitif + bonus kinerja
  • Asuransi kesehatan
  • Fleksibel work from home
  • Pelatihan dan pengembangan skill
  • Lingkungan kerja kolaboratif
  • Makan siang dan snack
